## ✨ What Hack.MSA Does

Hack.MSA helps young adults learn the things school never teaches:

- 💰 **Learn budgeting & financial literacy**
- 🧾 **Understand taxes and real-world systems**
- 🏠 **Navigate leases and contracts**
- 🎮 **Simulate real-life decisions**
- 📊 **Track progress and growth**
- 🤖 **Get AI-powered explanations**
- 🧠 **Build real-world decision-making skills**

---

## 🧠 Architecture — How it Works

Hack.MSA is built as a **multi-app monorepo**, combining mobile, web, and backend systems.

### 1️⃣ 📱 Adulting 101 — Mobile App  
A structured learning experience for real-world skills.

- Guided onboarding
- Two core tracks:
  - Money & Budgeting  
  - Taxes & Filing  
- Interactive lessons (quizzes, scenarios, checklists)
- Built-in tools:
  - Budget Builder  
  - Lease Checklist  
  - Tax Tracker  
- Progress tracking + reminders

---

### 2️⃣ 🎮 Adulting Simulator — Web App  
A decision-based simulation engine.

- Choose a character  
- Make real-life decisions  
- Experience consequences  
- Track stats:
  - Cash  
  - Stress  
  - Knowledge  
  - Score  
- Multiple endings  
- AI explains your choices in real time  

---

### 3️⃣ 🧩 LifeLevel — Gamified Learning  
A Duolingo-style experience for life skills.

- XP system  
- Streak tracking  
- Hearts (lives)  
- Weekly leaderboard  
- Bite-sized lessons  

---

### 4️⃣ ⚙️ API Server — AI + Backend  
The engine behind smart feedback.

- `/api/health` — system check  
- `/api/explain` — AI-powered explanations  

Handles:
- Decision analysis  
- AI responses  
- Data validation  
- Backend logic

## ⚡ Getting Started

Clone and install:

git clone https://github.com/SamiMiri/Hack.MSA.git  
cd Hack.MSA  
pnpm install  

---

## ▶️ Run the Apps

### 📱 Mobile App

cd artifacts/adulting-app  
pnpm start  

---

### 🎮 Simulator

cd artifacts/adulting-simulator  
pnpm dev  

---

### 🧩 LifeLevel

cd artifacts/lifelevel  
pnpm dev  

---

### ⚙️ API Server

cd artifacts/api-server  
pnpm dev  

---

## 🔐 Environment Variables

Create a `.env` file for the API:

AI_INTEGRATIONS_OPENAI_BASE_URL=  
AI_INTEGRATIONS_OPENAI_API_KEY=
