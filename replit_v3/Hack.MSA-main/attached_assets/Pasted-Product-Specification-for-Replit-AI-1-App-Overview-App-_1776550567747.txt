8. Simple Functional Description (For Engineers)
What the app does: A text-based story game that teaches adulting skills through interactive scenarios. Users make choices that lead to different outcomes, learn from consequences, and track progress through streaks and leaderboards.

Main screens/features:

Tutorial/Demo screen on first launch
Lesson/Scenario selection screen
Text-based story gameplay with decision points
Progress dashboard showing completed lessons and streak count
Leaderboard showing competition with friends
(Premium) Scenario creation tool
How users interact:

User launches app → completes tutorial → selects a lesson/scenario → reads situation and makes choices → sees consequences → completes scenario → earns streak progress → competes on leaderboard → returns weekly for new lessons
Premium users can create custom scenarios with level modifiers and fluid storytelling
Technical notes:

Focus MVP on one complete campaign/course as demo
Text-based progression (reference: "Example of Text-Based Story Progression - In progress...")
Mobile-first, game-like interface
Track streaks and unlock premium at 20-day streak or course completion
